COMP 125 - JavaScript Midterm Instructions

Introduction

Welcome to the COMP 125 JavaScript Midterm! In this exam, you will implement JavaScript functionalities based on the given questions. A working example is provided, refer to it to understand expected behavior.

You have 90 minutes to complete the exam.

General Guidelines

  1. Do Not Modify the HTML and CSS: You are only allowed to write JavaScript code within the provided script.js file.
  2. Understand Before Implementing: Carefully read each question and review the HTML and CSS files to find the correct IDs and structure.
  3. Use the Example: The working example demonstrates the expected functionality—use it as a guide.
  4. Attempt All Questions: Marks cannot be given for blank answers. If stuck for more than 10 minutes, move on and return later.
  5. Test Your ChatGPT Link: Before submitting, test your shared ChatGPT log link in incognito mode to ensure it works.

Midterm Questions

Each question is worth 10 marks (total: 70 marks).

Question 1: Hover Counters

  • Implement a function that tracks how many times a box has been hovered into and hovered out of.
  • Display both counters in real-time.

Question 2: Simple Comment System

  • Implement a comment system with:
    • A text input and a Post Comment button.
    • New comments appearing below previous ones.

Question 3: Typing Effect

  • Implement a function where a user enters a name, and when they click a button:
    • The name appears one character at a time.
    • There is a 500ms delay between each character appearing.

Question 4: Division Calculator (Try-Catch)

  • Implement a function that performs division between two numbers.
    • Use a try...catch block to handle division errors.
    • If the denominator is zero, throw an error and display an alert: “Cannot divide by 0”.

Question 5: Checkered Box Grid

  • The page contains a pre-defined 5x6 grid of empty squares.
  • Write JavaScript to apply alternating colors (blue and green) using a loop.

Question 6: Temperature Converter

  • Implement a function that converts temperatures between Celsius and Fahrenheit.
    • The conversion should update in real-time as the user types or changes the values with the arrows.
    • Ensure the output is rounded to one decimal place.

Question 7: Debugging Task - Vowel Counter

A pre-written JavaScript function is intended to count vowels in a given input but contains four errors that prevent it from working correctly.

  • Identify and fix the errors so that the function correctly counts vowels.
  • After fixing each error, describe in your own words what the mistake was on the line where you fixed it.
  • AI-generated explanations are not allowed. This is not an English class, your grammar doesn’t matter, but your explanation should show that you understand the issue.
