COMP 125 - JavaScript Final Exam Instructions

📌 Introduction

Welcome to the COMP 125 Final Exam. You have 90 minutes to complete this test. There are 7 independent JavaScript questions. Each is worth 10 marks. The exam will be marked out of 60. Your best 6 questions will be used.

You can reference this completed working example to help you understand what each question is supposed to do:

🧠 General Guidelines

  1. Only edit the .js file. Do not change any HTML or CSS.

  3. Use the working example to see how things should behave.

  4. Try all questions. If you’re stuck for a while, move on and return later.

  5. Use built-in browser tools like console.log() to test and debug.

🧪 Final Exam Questions (60 Marks Total)

Each question is worth 10 marks.

🔹 Question 1 – To-Do List

Make the “Add” button add an item to the list. Make the “Clear List” button remove all items. The list must be saved using localStorage so it remains after refresh.

🔹 Question 2 – Pokémon Finder

When the user enters a number between 1 and 1024 and clicks “Get Pokémon”, fetch and show the shiny official image for that Pokémon using the PokéAPI.

Hint: The image you’re looking for is located within the sprites object, under other.official-artwork.

🔹 Question 3 – Typing Speed Test

When the user clicks “Go”, they should type the sentence:

the quick brown fox jumps over the lazy dog

They have 10 seconds. Show a timer countdown while they type. If they finish in time, show “You win!”. Otherwise, show “You lose.”

🔹 Question 4 – Holiday Countdown

Populate the dropdown using JavaScript (not HTML) with Halloween, Christmas, and New Year’s Eve (2025). When a user selects a holiday, display how many days remain until that date.

🔹 Question 5 – Library Management

After uploading a JSON file, let users enter an ISBN and check books in or out. Show a message for success, or if no copies are available. Use a Book class to manage logic.

🔹 Question 6 – Code and Username Validator

Check a form with two fields:

  • Secret Code – Must be “COMP125”

  • Username – Must contain only letters and numbers (use a regex)

Use setCustomValidity() to show error messages as the user types. Let the browser handle validation and submission normally.

🔹 Question 7 – Book Sorting Challenge

After uploading a .json file, sort the books by:

  1. availableCopies (highest first)

  2. title length (shortest first)

Show the top 10 books like:

The Hobbit (4)
1984 (5)
