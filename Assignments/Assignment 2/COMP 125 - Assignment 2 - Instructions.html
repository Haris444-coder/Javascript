Requirements

1️⃣ Correct Guess Handling (10%)

  • Players can click a letter or press a key on their keyboard.
  • If the letter is in the word, reveal all occurrences of that letter.
  • If the letter is incorrect, it should not be added.

2️⃣ Lives System (10%)

  • The game starts with 6 lives.
  • Every incorrect guess reduces lives by 1.
  • Display the number of lives remaining.

3️⃣ Win/Loss Conditions (10%)

  • The player wins when all letters are revealed.
  • The player loses when lives reach 0.
  • Display an appropriate win or game-over message.

4️⃣ Keyboard & Click Input Support (10%)

  • Players should be able to click the letter buttons on screen.
  • Players should also be able to press keys on their keyboard (lowercase).
  • Disable used letters so they can’t be clicked or typed again.

5️⃣ Time Mode (10%)

  • The game should have a 30-second countdown timer.
  • If time runs out before the word is guessed, the game ends.
  • The timer should be displayed on screen.
  • The timer starts when the first guess is made.
