Requirements

  1. Static Controls (15%)

    • Implement buttons to turn on the Red, Yellow, and Green lights.
    • Only one light should be active at a time.
    • If a mode (Auto or Random) is running, these buttons should stop it.

  2. Auto Mode (15%)

    • Implement the Auto Mode button to cycle through the predefined autoModeSequence array at 500ms intervals.
    • The sequence should reset to the beginning after reaching the end of the array.

  3. Random Mode (20%)

    • Implement the Random Mode button to change the light to a random color (red, yellow, or green) every 500ms.
    • Add functionality to stop this mode when the Stop Random button is clicked.

  4. ChatGPT Usage (50%)

    • Refer to the ChatGPT usage rubric provided in the outline.

Hints

  • You will need to add the active class programmatically to the appropriate light color for it to turn on.
  • Use the autoModeSequence array already defined in the script.js file for cycling through colors in Auto Mode:
    const autoModeSequence = [
  "red", "yellow", "green", "yellow", "red", "yellow", "green",
  // Continue sequence...
];
  • Use setInterval to repeatedly execute a function for both Auto Mode and Random Mode. Use clearInterval to stop the interval when switching modes or stopping the lights.
