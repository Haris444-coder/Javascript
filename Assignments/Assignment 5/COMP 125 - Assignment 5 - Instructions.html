COMP 125 - Assignment 5: Pokédex Search + Favourites

✨ Objective

Your task is to build a dynamic Pokédex application using JavaScript. Users can search for Pokémon by name or ID, view their details in a styled card, open a modal to view more information, and add up to 6 Pokémon to a favourites list stored in localStorage. You will work with an external API (PokeAPI).

You will be provided with:

  • A completed index.html
  • A completed styles.css
  • An empty script.js where all your work goes
  • A Pokédex logo and loading spinner image

You may only edit script.js.

A working example of the completed assignment is available here:
📍 https://soft-pothos-9c0c9e.netlify.app

🔍 Features & Requirements

  1. Search Functionality (15%)

    • User enters a Pokémon name or ID and clicks “Search Pokémon”
    • Fetches data from PokeAPI
    • While loading, a Pokéball spinner is shown
    • If the Pokémon is not found, an error message is displayed

  2. Display Basic Info (15%)

    • Show a styled card with:
      • Name
      • ID
      • Image
      • Types
      • Abilities
      • A “More Info” button
      • An “Add to Favourites” button

  3. "More Info" Modal (10%)

    • Clicking “More Info” opens a modal showing:
      • Height, weight, base stats
      • Modal can be closed with the X or by clicking outside the box

  4. Favourites System (10%)

    • Clicking “Add to Favourites” saves the Pokémon to localStorage
    • Users can only favourite up to 6 Pokémon
    • Clicking “View Favourites” toggles between favourites and search

  5. ChatGPT Usage (50%)

    • This assignment emphasizes using ChatGPT to debug and learn
    • Submit your ChatGPT conversation link(s) as a comment at the top of script.js
