COMP 125 - Assignment 4: Pixar Movie Gallery

✨ Objective

Your task is to create a dynamic web gallery to showcase Pixar movies using JavaScript. You will load movie data from a file, filter and search through it, and display each movie as a visual card. You will also apply a custom animation when users hover over movie cards.

Your task is to create a dynamic web gallery to showcase Pixar movies using JavaScript. You will load movie data from a file, filter and search through it, and display each movie as a visual card. You will also apply a custom animation when users hover over movie cards.

You will be provided with:

  • A completed index.html
  • A mostly completed styles.css (just missing the hover animation)
  • A starter script.js file with variables and empty event listeners

You may only edit script.js, except for one specific CSS requirement: you must add your own CSS hover animation to the .movie class. This can be any animation you like, as long as it’s triggered by hovering.

A working example of the completed assignment is available here:
👉 https://endearing-yeot-c8809d.netlify.app/

🔍 Features & Requirements

  1. File Upload (10%)

    • Allow the user to upload a .json file.
    • Load the movie data

  2. Movie Display with Animation (20%)

    • Render each movie as a poster card with title, director, and year.
    • Add a CSS animation on hover to make the movie card move, scale, or animate. You can use any animation you like, it does not need to match the working example exactly. Be creative!
    • This is the only place you can modify the CSS.

  3. Filter Controls (10%)

    • Dropdowns to filter by year and director.
    • Dropdown to sort by ascending/descending order by release date.
    • Filters should work simultaneously.
    • If the user starts filtering, the search input should reset.

  4. Search Input (10%)

    • A text input that filters movies by title as the user types.
    • If the user starts searching, filters should reset.

  5. ChatGPT Usage (50%)

    • This assignment heavily emphasizes your ability to use ChatGPT effectively.
    • Submit your shared ChatGPT conversation link(s) as a comment at the top of your script.js file.

💡 Hints

  • Check the HTML Carefully
    Inspect the working example in the browser using developer tools. Open the Elements tab and explore how each movie card is structured. Pay attention to class names and layout so you can recreate the same structure using JavaScript.

  • Don’t Forget to Reset the Opposite Input
    If the user applies a filter, clear the search box. If they start typing in the search box, reset all the filters. You should do this inside your event listeners.

  • Use .filter() and .includes()
    When filtering arrays, the .filter() method is your friend. Use .includes() (with .toLowerCase()) to check if a movie title matches the user’s search.

  • Be Creative with CSS Animation
    You can animate on hover however you like—scale, rotate, bounce, pulse, etc. Just keep the animation inside your CSS file and apply it to .movie:hover.
